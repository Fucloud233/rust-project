# rust-project

本插件能够在不创建Cargo项目的条件下成功激活rust-analyzer插件。

运行原理：https://rust-analyzer.github.io/manual.html#non-cargo-based-projects

## Features

* 在不创建Cargo项目时，也能正常使用rust-analyzer插件
* 可以自定义选择配置`rust-project.json`的方式

## Requirements

请先安装好`rust-analyzer`插件。
